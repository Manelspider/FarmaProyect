# FarmaNexo API

## Descripción

Proyecto Django para la API de FarmaNexo. Proporciona endpoints REST para la gestión de recetas, usuarios y seguimiento de estados de recetas. Diseñada para integrarse con el panel web y aplicaciones móviles.

## Características principales

- Gestión de usuarios médicos y proveedores.
- CRUD de recetas médicas.
- Endpoints REST seguros y escalables.
- Conexión a PostgreSQL como base de datos.
